# About
Simple cli to generate new C or C++ projects.

# Usage
     
**Generate new project** 

```
cgen new [-c | -cpp] [project name]
```
     
**Display help page**
```
cgen [-h | --help]
```
     
**Get version**
```
cgen [-v | --version]
```

# Installation
Copy the `cgen` file in `cgen/bin/` to `/usr/bin/`
